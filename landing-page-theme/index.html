About Landing-Page

Landing-Page is a free Bootstrap 3 theme created by Start Bootstrap. It can be yours right now, simply download the
template on the Preview page. The theme is open source, and you can use it for any purpose, personal or commercial.

The Jekyll version is brought to you by Shane Weng

This theme features stock photos by to the Stock Photo.

Landing-Page includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization.
